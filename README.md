# Employee-Salary-Satisfaction-Attrition-Analysis

📌 Project Title:

Employee Salary, Satisfaction & Attrition Analysis

📝 Description:

This project focuses on analyzing employee data to uncover insights related to salary prediction, job satisfaction, and employee attrition. The dataset contains detailed information about employees, including demographics, skills, compensation, expenses, and work conditions.

The project aims to:
Predict employee salaries using machine learning models
Identify factors affecting job satisfaction
Analyze employee attrition trends
Explore relationships between skills, experience, and compensation

🎯 Objectives:
Build regression models to predict salary
Develop classification models for attrition prediction
Perform exploratory data analysis (EDA)
Extract actionable business insights

⚙️ Tech Stack:
Python
Pandas, NumPy
Matplotlib, Seaborn
Scikit-learn

📊 Key Features:
Data Cleaning & Preprocessing
Feature Engineering
Model Training (Linear Regression, Ridge, Lasso, Logistic Regression)
Model Evaluation (RMSE, Accuracy, etc.)
Insightful Visualizations

🚀 Outcomes:
Identified key factors influencing salary and job satisfaction
Built predictive models for business decision-making
Provided insights into employee financial behavior
